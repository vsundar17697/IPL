Instructions : 

1. Create a main folder , with the name '<NAME>' as desired, and then create a sub-folder within it, named 'WEB-INF' and also an index.html or home.html named file, which renders the HTML.

2. Within the 'WEB-INF' folder, create another folder called 'classes' which contains the java file, that the servlet performs.

3. Within the 'WEB-INF' folder, there is also a file named 'web.xml' which contains the xml code for the sevlet, which names the servlet and also assigns a class to it, and maps it to a URL.

4. The server needs to be run each time any changes are made in the java file.

In this web-app, Session Tracking mechanism using HttpSessionObject has been implemented.
